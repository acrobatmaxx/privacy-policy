Privacy Policy

This application is used exclusively by the owner to integrate Google Nest devices with Home Assistant for personal home automation purposes.

Data Access and Usage

We request access to the following Google account information and services:

  • openid, userinfo.email, userinfo.profile: Used to identify the account holder and associate the integration with their Google account.
  • sdm.service, sdm.device.control.service, sdm.thermostat.service, nest-account: Used to access and control the user's Nest thermostat.
  • pubsub: Used to receive real-time updates from the Nest device via Google Cloud Pub/Sub.

No data is stored, shared, or transmitted to third parties. All data access is limited to the authenticated user and used solely for device control and automation within Home Assistant.

Data Retention

No personal data is retained. All interactions occur in real time and are processed locally through Home Assistant.

Security

This application uses OAuth 2.0 for secure authentication. No credentials or sensitive data are stored externally.
